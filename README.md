# Instruction  

## Quick start  

- Create database based on your environment (eg. development, production)  
- Create `.env` file. For file structure please follow [this section](#env-structure)
- Run `yarn bootstrap`  
` Run `yarn dev`  

## .env structure  

```js
TOKEN_SECRET=
ENV=

DB_HOST=
DB_PORT=
DB_USERNAME=
DB_PASSWORD=
DB_DATABASE=
```
